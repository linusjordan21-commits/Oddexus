ODDEXUS BOT — så här kommer du igång
=====================================

Du behöver ett AKTIVT medlemskap på oddexus.com. Boten loggar in med ditt
oddexus.com-konto och fungerar bara så länge ditt medlemskap är aktivt.

STARTA (Mac)
------------
1. Dubbelklicka  "Starta-boten.command".
   (Första gången installeras allt som behövs — det tar några minuter.)
2. Skriv in ditt användarnamn + lösenord från oddexus.com när det frågas.
   (Du loggar bara in första gången — sedan kommer den ihåg dig.)
3. Följ stegen i fönstret (logga in på casinona, kalibrera SPIN + SALDO).

VARJE MÅNAD
-----------
Du behöver INTE ladda ner något nytt. Så länge du betalar fungerar samma bot.
Slutar medlemskapet gälla säger boten till och startar inte förrän du förnyat
på oddexus.com.

KRAV
----
- Internet (boten kontrollerar ditt medlemskap varje gång den startar).
- Mac med Python 3.9+ (installeras automatiskt om det saknas via Homebrew).

TANGENTER NÄR BOTEN KÖR
-----------------------
  €   = starta / pausa / fortsätt (ett tryck, ingen Enter)

Problem? Hör av dig till oss på oddexus.com.
